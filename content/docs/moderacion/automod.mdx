# Auto-moderación

El sistema de auto-mod actúa de forma autónoma para mantener el orden en tu servidor.

## Filtros disponibles

- **Anti-spam** — Detecta mensajes repetidos o en ráfaga.
- **Anti-links** — Bloquea URLs no permitidas.
- **Anti-palabras** — Filtra palabras o frases específicas.
- **Anti-mayúsculas** — Limita mensajes en caps lock.
- **Anti-menciones** — Previene el mention spam.

## Configuración

```
/automod configurar
```

Abre el panel interactivo para ajustar cada filtro, establecer acciones (eliminar, warn, timeout) y definir canales o roles exentos.
