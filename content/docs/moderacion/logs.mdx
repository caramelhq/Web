# Configurar logs

Los logs registran automáticamente acciones del servidor en canales que tú elijas.

## Tipos de log

- **Moderación** — Sanciones, bans, kicks, warns.
- **Mensajes** — Mensajes editados y eliminados.
- **Miembros** — Entradas, salidas y cambios de nickname.
- **Roles** — Cambios de roles en miembros.
- **Canales** — Creación, edición y eliminación de canales.
- **Voz** — Entradas y salidas de canales de voz.

## Configuración

```
/logs configurar
```

Selecciona el tipo de log y el canal de destino desde el panel interactivo.
