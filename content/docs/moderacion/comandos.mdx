# Comandos de moderación

## Sanciones

| Comando | Descripción |
| --- | --- |
| `/ban @usuario [razón]` | Expulsa permanentemente a un usuario |
| `/unban @usuario` | Revoca el ban de un usuario |
| `/kick @usuario [razón]` | Expulsa del servidor sin ban |
| `/timeout @usuario [duración]` | Silencia temporalmente a un usuario |
| `/warn @usuario [razón]` | Añade una advertencia al historial |

## Mensajes

| Comando | Descripción |
| --- | --- |
| `/purge [cantidad]` | Elimina mensajes en masa |
| `/purge @usuario [cantidad]` | Elimina mensajes de un usuario específico |

## Gestión de roles

| Comando | Descripción |
| --- | --- |
| `/role add @usuario @rol` | Asigna un rol |
| `/role remove @usuario @rol` | Retira un rol |
