# Sistema de casos

Cada sanción aplicada genera un caso con ID único que puedes consultar, editar y cerrar.

## Comandos

| Comando | Descripción |
| --- | --- |
| `/caso ver [id]` | Consulta los detalles de un caso |
| `/caso lista @usuario` | Lista todos los casos de un usuario |
| `/caso editar [id] [razón]` | Edita la razón de un caso existente |
| `/caso borrar [id]` | Elimina un caso del registro |
| `/historial @usuario` | Resumen del historial de sanciones |
