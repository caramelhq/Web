# Moderación

El módulo de moderación de Caramel te da control total sobre tu servidor: desde sanciones manuales hasta sistemas de auto-moderación avanzados.

## Funciones principales

- **Comandos de moderación** — Ban, kick, timeout, warn y más.
- **Auto-moderación** — Filtra spam, links y palabras prohibidas automáticamente.
- **Sistema de casos** — Registro de todas las sanciones aplicadas.
- **Logs de moderación** — Historial detallado de acciones en tu servidor.

## Activar el módulo

```
/modulo activar moderacion
```
