# Multiplicadores

Los multiplicadores aumentan la cantidad de XP que los usuarios ganan en canales o con roles específicos.

## Por canal

```
/niveles multiplicador canal #canal [factor]
```

Ejemplo: `/niveles multiplicador canal #general 2` → doble XP en ese canal.

## Por rol

```
/niveles multiplicador rol @rol [factor]
```

Los usuarios con ese rol ganan XP multiplicado en todo el servidor.

## Ver multiplicadores activos

```
/niveles multiplicadores
```
