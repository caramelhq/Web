# Roles de recompensa

## Añadir recompensa

```
/niveles recompensa añadir [nivel] @rol
```

Cuando un usuario alcanza ese nivel, Caramel le asigna el rol automáticamente.

## Eliminar recompensa

```
/niveles recompensa eliminar [nivel]
```

## Listar recompensas

```
/niveles recompensas
```

## Modo acumulativo vs. único

- **Acumulativo** — El usuario conserva todos los roles de niveles anteriores.
- **Único** — Solo mantiene el rol del nivel más alto alcanzado.

```
/niveles modo acumulativo
/niveles modo unico
```
