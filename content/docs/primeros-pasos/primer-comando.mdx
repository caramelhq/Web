# Primer comando

## Explorar comandos

Escribe `/ayuda` en cualquier canal donde Caramel tenga acceso. Aparecerá un menú interactivo con todos los módulos disponibles.

## Comandos básicos de administración

```
/configurar          → Abre el panel de configuración del servidor
/modulo activar      → Activa un módulo específico
/modulo desactivar   → Desactiva un módulo
/info                → Información general del bot
/ping                → Comprueba la latencia
```

## Próximos pasos

Una vez que hayas ejecutado los comandos básicos, te recomendamos:

1. Revisar la sección de [Moderación](/docs/moderacion) si quieres gestionar tu comunidad.
2. Configurar [Música](/docs/musica) si tu servidor es activo en voz.
3. Activar [Economía](/docs/economia) o [Niveles](/docs/niveles) para gamificar la participación.
