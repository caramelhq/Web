# Permisos necesarios

Caramel solicita permisos de administrador durante la instalación para garantizar el correcto funcionamiento de todos sus módulos.

## Permisos recomendados

| Permiso | Para qué se usa |
| --- | --- |
| Administrador | Acceso completo a todas las funciones |
| Gestionar canales | Crear y editar canales de tickets |
| Gestionar roles | Asignar roles de nivel y economía |
| Enviar mensajes | Responder a comandos |
| Leer historial | Moderar mensajes anteriores |
| Conectar / Hablar | Reproducir música en canales de voz |

<Callout type="warning">
  Si prefieres no dar permisos de administrador, asegúrate de asignar manualmente cada permiso necesario para los módulos que uses.
</Callout>
