# Primeros pasos

Esta sección cubre todo lo que necesitas para tener Caramel funcionando en tu servidor desde cero.

## Pasos rápidos

1. **Invitar el bot** — Añade Caramel a tu servidor con los permisos necesarios.
2. **Configurar permisos** — Asegúrate de que Caramel tiene acceso a los canales correctos.
3. **Activar módulos** — Habilita solo los módulos que quieras usar.
4. **Primer comando** — Ejecuta `/ayuda` para ver todos los comandos disponibles.

Continúa leyendo para una guía detallada de cada paso.
