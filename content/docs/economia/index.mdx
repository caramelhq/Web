# Economía

El módulo de economía añade una capa de gamificación a tu servidor con moneda propia, tienda y trabajos.

## Funciones principales

- **Moneda personalizada** — Define el nombre y símbolo de tu moneda.
- **Banco y billetera** — Los usuarios gestionan su propio dinero.
- **Tienda** — Vende roles y recompensas con la moneda del servidor.
- **Trabajos** — Los usuarios ganan dinero completando trabajos.
- **Transferencias** — Los usuarios pueden enviarse dinero entre sí.

## Activar el módulo

```
/modulo activar economia
```
