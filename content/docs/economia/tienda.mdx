# Gestionar tienda

## Añadir ítems

```
/tienda añadir [nombre] [precio] [tipo]
```

**Tipos disponibles:**
- `rol` — Asigna un rol al comprador.
- `personalizado` — Notifica a los admins para entrega manual.

## Eliminar ítems

```
/tienda eliminar [id]
```

## Ver la tienda

Los usuarios pueden ver la tienda con:

```
/tienda
```
