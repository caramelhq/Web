# Sistema de trabajos

Los trabajos permiten a los usuarios ganar moneda de forma activa. Cada trabajo tiene un cooldown y una recompensa configurable.

## Trabajos predeterminados

Caramel incluye varios trabajos de ejemplo: programador, chef, repartidor, minero, etc.

## Crear un trabajo personalizado

```
/trabajo crear [nombre] [pago-min] [pago-max] [cooldown-horas]
```

## Gestión

```
/trabajo listar
/trabajo eliminar [nombre]
```

Los usuarios ejecutan `/trabajar` para elegir su trabajo y reclamar su salario.
