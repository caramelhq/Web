# Configurar moneda

## Nombre y símbolo

```
/economia moneda nombre [nombre]
/economia moneda simbolo [símbolo]
```

Ejemplo: `/economia moneda nombre Caramelos` y `/economia moneda simbolo 🍬`

## Saldo inicial

Define cuánta moneda reciben los nuevos miembros al unirse:

```
/economia inicio [cantidad]
```

## Recompensa diaria

Configura el comando `/daily` para que los usuarios reclamen su bonificación diaria:

```
/economia daily [cantidad] [cooldown en horas]
```
