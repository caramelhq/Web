# Comandos de música

## Reproducción

| Comando | Descripción |
| --- | --- |
| `/play [canción o URL]` | Añade una canción a la cola |
| `/pause` | Pausa la reproducción |
| `/resume` | Reanuda la reproducción |
| `/stop` | Detiene la reproducción y limpia la cola |
| `/skip` | Salta a la siguiente canción |
| `/skipto [posición]` | Salta a una posición específica de la cola |

## Cola

| Comando | Descripción |
| --- | --- |
| `/queue` | Muestra la cola actual |
| `/shuffle` | Mezcla la cola aleatoriamente |
| `/loop` | Activa o desactiva el bucle |
| `/remove [posición]` | Elimina una canción de la cola |

## Control de volumen

| Comando | Descripción |
| --- | --- |
| `/volume [0-200]` | Ajusta el volumen |
| `/bass [nivel]` | Ajusta el efecto de bajos (Premium) |
