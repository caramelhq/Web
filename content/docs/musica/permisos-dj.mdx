# Permisos de DJ

El sistema de DJ te permite definir qué roles pueden controlar la música.

## Configuración

```
/dj configurar @rol
/dj eliminar @rol
/dj lista
```

## Modo DJ estricto

Con el modo estricto activado, solo los DJs pueden usar `/play`, `/skip`, `/stop` y `/volume`. Los demás usuarios solo pueden votar para saltar canciones con `/voteskip`.

```
/dj estricto activar
/dj estricto desactivar
```
