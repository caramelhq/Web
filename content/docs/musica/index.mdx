# Música

El módulo de música de Caramel soporta múltiples fuentes de audio con cola ilimitada (Premium) y filtros de audio avanzados.

## Características

- **Múltiples fuentes** — YouTube, Spotify, SoundCloud y más.
- **Cola de reproducción** — Gestiona la cola con comandos intuitivos.
- **Filtros de audio** — Bass boost, nightcore, vaporwave y más (Premium).
- **Control de DJ** — Restringe los comandos a roles específicos.

## Activar el módulo

```
/modulo activar musica
```

<Callout type="info">
  La cola está limitada a 5 canciones en el plan gratuito. Actualiza a Premium para desbloquear la cola ilimitada.
</Callout>
