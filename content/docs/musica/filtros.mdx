# Filtros de audio

Los filtros de audio están disponibles en el plan **Premium**. Se aplican en tiempo real sin interrumpir la reproducción.

## Filtros disponibles

| Filtro | Descripción |
| --- | --- |
| `bassboost` | Aumenta los graves |
| `nightcore` | Aumenta el pitch y tempo |
| `vaporwave` | Disminuye el pitch y tempo |
| `8d` | Efecto de audio 8D |
| `karaoke` | Reduce las voces |
| `vibrato` | Efecto de vibrato |
| `tremolo` | Efecto de trémolo |

## Uso

```
/filter [nombre del filtro]
/filter off
```
