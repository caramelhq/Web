# Bienvenido a Caramel Docs

Caramel es un bot de Discord todo en uno diseñado para adaptarse a tu servidor. Estas guías te ayudarán a sacarle el máximo partido a todos sus módulos.

## ¿Por dónde empiezo?

Si es tu primera vez con Caramel, comienza por la sección de [Primeros pasos](/docs/primeros-pasos).

## Módulos disponibles

| Módulo | Descripción |
| --- | --- |
| [Moderación](/docs/moderacion) | Gestiona tu comunidad con herramientas profesionales |
| [Música](/docs/musica) | Reproduce música desde múltiples fuentes |
| [Economía](/docs/economia) | Sistema de moneda, tienda y trabajos |
| [Niveles](/docs/niveles) | XP, niveles y roles de recompensa |
| [Tickets](/docs/tickets) | Soporte interno con transcripciones |

## Buscar en la documentación

Usa la barra de búsqueda en la parte superior o presiona <kbd>Ctrl</kbd> + <kbd>K</kbd> para buscar cualquier tema.
