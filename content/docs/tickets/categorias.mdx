# Categorías

Las categorías permiten segmentar los tickets según su propósito.

## Crear categoría

```
/tickets categoria crear [nombre]
```

## Configurar roles de staff por categoría

```
/tickets categoria staff [nombre] @rol
```

Solo los usuarios con ese rol podrán ver los tickets de esa categoría.

## Eliminar categoría

```
/tickets categoria eliminar [nombre]
```

## Listar categorías

```
/tickets categorias
```
