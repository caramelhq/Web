# Tickets

El módulo de tickets crea canales privados para gestionar solicitudes de soporte de forma organizada.

## Funciones principales

- **Panel de tickets** — Botón interactivo para que los usuarios abran tickets.
- **Categorías** — Organiza los tickets por tipo (soporte, reporte, etc.).
- **Transcripciones** — Genera un log en HTML o texto al cerrar un ticket.
- **Roles de staff** — Define qué roles pueden ver y gestionar tickets.

## Activar el módulo

```
/modulo activar tickets
```
